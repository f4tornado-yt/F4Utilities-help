F4Utilities

F4Utilities is a general utilities extension.
It currently does basic accounting, like calculating profits,
and password management.
I love suggestions, so send them to me here!

  • Money

  • tax - calculates tax

    =calcTax(price, percent, round)

    price - original price (number)
    percent - tax percent (number) (input 6, not 0.06)
    round - If true, it will round to the nearest hundredth (boolean)


  • sale - calculates price of an item on sale

    =calcSale(price, percent, round)

    price - the original price (number)
    percent - percent off (number) (input 6, not 0.06)
    round - If true, it will round to the nearedt hundredth (boolean)



  • Profit - sorts events or programs by profitability
    select a range, like this,

    the left side has the name of a program or event, the right side is the profits from the program or event.
    Then go to the F4Utilities tab and click profit.

    You can also add info about the event to the right of the profit generated, make sure to select that too.

  • Password Management

  • Generate Password

    =makePass(chars, cap-chars, nums, symbols)

    chars - number of lowercase letters (number)
    cap-chars - number of uppercase letters (number)
    nums - number of numbers (number)
    symbols - number of symbols (number)
